Universal Timeline Manager

This plugin creates chronological playlists for cinematic universes based on your JSON configuration file.

Configuration File

Location: /config/timeline_manager_config.json

A default configuration file has been created automatically on first install with an example timeline.

Edit this file to add your own universes and items.

Creating Playlists

Use the API endpoint to create playlists from your configuration:

POST /Timeline/CreatePlaylists?userId=YOUR_USER_ID

See the plugin README for detailed instructions and examples.

API Endpoints

  • GET /Timeline/Test - Test if plugin is working
  • GET /Timeline/Config - Get current configuration
  • POST /Timeline/Validate - Validate configuration
  • POST /Timeline/CreatePlaylists - Create playlists
